About Doc-Auditor

Doc-Auditor is a compliance automation platform built to streamline the auditing process for accounting and administrative teams. By utilizing regular expressions and PDF parsing heuristics, it guarantees data accuracy against customized requirements.

How It Works

  • 1. File Ingestion: Securely uploads PDF invoices into internal systems.
  • 2. Text Extraction: Converts vectorized and OCR paths into raw textual string data via pdfplumber.
  • 3. Rule Engines: Locates GST IDs, Invoice metadata, and Total amounts using RegEx.
  • 4. Validation Logs: Saves historical data inside a secure SQLite database for audit trails.

Technology Stack

Powered by Python, Flask, SQLite, and a zero-dependency futuristic CSS architectural model.
